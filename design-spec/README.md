## Learnovo Design & Product Spec

This package contains design tokens, reusable component examples, API contracts, and product flow specs to guide UI/UX and SaaS readiness.

- tokens/: Color, typography, spacing, and Tailwind extension tokens
- components/: React + TypeScript examples (KPI cards, filters, charts)
- api/: OpenAPI spec for core resources and billing
- specs/: Functional specs (onboarding, billing, CSV, RBAC, etc.)

Use these as the single source of truth for visual and interaction guidelines across Admin, Teacher, Student, Parent, and SuperAdmin roles.
