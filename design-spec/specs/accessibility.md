### Accessibility (WCAG 2.2 AA)

- Color contrast: text ≥ 4.5:1, large text ≥ 3:1
- Keyboard: visible focus, tab order, skip links
- Forms: labels, descriptions, error text, aria-live for validation
- Components: roles/aria as needed; no keyboard traps
- Motion: reduce motion preference respected
- Testing: axe-core, keyboard-only walkthroughs, screen reader checks
